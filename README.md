# SMPTE ST 2117-10

SMPTE ST 2117-10 maps the bitstream for ST2117-1 VC-6 into the MXF Generic Container. 
A Committee Draft of SMPTE RP 268-3 was previously made available here for a public review period, which ended on 2023-12-31.
